# 辰東《遮天》完整資料研究

> 遮天三部曲第一部，主角葉凡。連載於起點中文網，2010年10月14日至2013年5月21日，共1822章，636.14萬字。

## 一、核心劇情線（按時間順序）

### 1. 九龍拉棺事件（地球篇 / 開篇）

葉凡與大學同學在泰山聚會，偶然發現五色祭壇，九具龐大龍屍拉著一口青銅古棺降臨。眾人被吸入九龍拉棺，先被帶到熒惑（火星），在火星上的大雷音寺中拿到許多古器（葉凡獲得綠銅塊/菩提子），但因古器被取走，導致大雷音寺鎮壓的鱷祖出世，追殺眾人，死傷多名同學。眾人逃回九龍拉棺，最終被帶到北斗星域東荒七大生命禁區之一的荒古禁地外圍。

### 2. 東荒初期篇

- 眾同學在荒古禁地外吃下多顆聖果，葉凡被發現擁有「荒古聖體」體質
- 遇到女修士薇薇，被帶到靈墟洞天，了解北斗五大區域（東荒、西漠、南嶺、北原、中州）
- 龐博因服用聖果被靈墟洞天視為仙苗
- 東荒妖帝墓浮現，葉凡與龐博前往。龐博被青帝十九代孫的大妖附身離去，葉凡奪寶後離開靈墟洞天
- 葉凡認識姜老伯與姜婷婷（姜家流落在外的族人），後姜家接回老伯祖孫
- 姜逸晨覬覦葉凡寶物追殺，葉凡逃亡

### 3. 太玄派與東荒聖地篇

- 葉凡借太玄門收徒之際入門，拜李若愚為師
- 被姬紫月帶到太玄門，二人經歷青銅古殿歷險（狠人大帝遺跡）
- 在李若愚悟道之際，葉凡借助菩提子領悟九秘之一「皆」字秘
- 太玄門遭逢孔雀王之亂，姬紫月被追殺，葉凡護送她回姬家
- 與搖光聖地、瑤池聖女產生交集與衝突
- 進入太初古礦禁區冒險，與搖光聖子、瑤池聖女同行
- 在東荒北域曲州雲斷山脈，鎮殺姬家、搖光聖地子弟，橫壓年輕一代

### 4. 北域源區篇

- 葉凡被搖光聖地的人帶往北域挖源，在礦區騷亂中逃出
- 學習源術，開始展露源術天賦
- 遇到段德（表面是猥瑣的盜墓人，實則為九世輪迴的曹雨生 / 渡劫天尊）
- 遇到黑皇（大黑狗）-- 無始大帝救活並封在紫山中的忠犬，被葉凡帶出
- 黑皇帶葉凡找到瑤池故地，協助擊殺姬家修士
- 與龐博重逢，合力鎮壓附身龐博的老妖神識，龐博取得妖帝古經和青蓮法寶

### 5. 南嶺篇

- 葉凡前往南嶺，了解妖族勢力
- 涉及戰神殿、妖皇殿、蠻族等勢力
- 結識秦瑤（妖族女子，紅顏知己，後因修煉入岔而死）

### 6. 中州篇

- 葉凡使用黑皇的符文穿越到中州
- 救了雨蝶公主，獲贈荒廬領地（毗鄰奇士府）
- 進入奇士府（萬載開啟一次，只收聖子級強者），突破至仙台境界
- 中州勢力格局：大夏皇朝、古華皇朝、神州皇朝、羽化神朝、陰陽教等
- 進入萬妖嶺古樹林，得到「太皇經」化龍卷
- 結識安妙依（妙欲庵傳人，與葉凡產生真情，後遠走西漠修佛）
- 與中州各大勢力周旋，聲名鵲起

### 7. 西漠篇

- 西漠以佛門為主，須彌山大雷音寺為聖地
- 阿彌陀佛大帝曾坐鎮，帝器降魔杵
- 釋迦摩尼從地球走出到北斗西漠，被認為是阿彌陀佛轉世
- 葉凡與西漠佛門的交集

### 8. 紫微星域篇

- 葉凡離開北斗前往紫微星域
- 參加狼神壽宴，斬殺金烏族太子，與金烏族結仇
- 肉搏紫微第一天女伊輕舞
- 擊殺金烏族十大聖主級太子、大九頭蛟龍、太陰神子、赤陽三奇才
- 大戰大成王者投影，留下傳奇後離去
- 紫微星域無帝器、無不死藥，全都去了葬帝星（北斗）

### 9. 回歸地球篇

- 葉凡離去二十多年後回到地球（約2030年代）
- 得知父母已故去
- 在地球收了弟子
- 尋找崑崙山、地球古蹟的修行秘密
- 斬道成功，帶張青陽等人西出函谷關，回到北斗

### 10. 成聖之路

- 葉凡突破至聖人境界，但天地不認可聖體，降下先天道圖壓制
- 姜太虛捨身消耗自身神靈血為葉凡破解先天道圖
- 聖體的詛咒源於神話時代，地府至尊用秘術詛咒聖體一族
- 葉凡吃了整顆九妙不死藥，化開苦海並打破四極詛咒

### 11. 準帝之戰

- 帝路五關，葉凡擊殺搖光聖子、擊退尹天德
- 宇宙邊荒帝子戰場，大戰不死天皇座下金色戰神念身
- 擊殺多位準帝級強者（神墟禁區雷神、仙陵禁區神冥、黃金天女、萬龍巢龍女、輪迴海少主）
- 晉升準帝六重天，無敵年輕一代

### 12. 證道成帝

- 葉凡以力證道，將諸天萬道踩在腳下，打破聖體不能成帝的詛咒
- 號「葉天帝」，建立天庭
- 天心印記不與天道融合，而是凌駕於天道之上

### 13. 平定禁區（成帝後）

- 圓滿百年，聖體大成，沐浴霸體之血
- 擊殺至尊，平定霸體祖洞
- 平滅輪迴海、不死山兩大禁區
- 於飛仙古星殺四大至尊
- 再平仙陵、神墟禁區
- 上蒼（葬天島）、太初古礦兩禁區被迫自世間消失
- 唯一保留狠人大帝所在的荒古禁地

### 14. 成仙路開啟與黑暗動亂

- 仙域裂縫顯現，成仙路開啟
- 禁區至尊從仙源中走出，爭奪成仙資格
- 成仙計畫失敗後，至尊們發動史上最慘烈的黑暗動亂
- 幾大生命古星被毀，生靈塗炭
- 瑤池萬族盛會之戰
- 荒古禁地第一代主人力拼長生天尊，同歸於盡
- 黃帝、姬子、神農、姜太虛、蓋九幽分別與輪迴之主、光暗至尊、神墟之主同歸於盡
- 葉天帝率黃金時代眾天驕征戰禁區，黑暗動亂成為歷史

### 15. 九世蛻變與紅塵為仙

| 世 | 關鍵事件 |
|----|----------|
| 第一世 | 證道成帝，建天庭。晚年借麒麟不死藥開啟第二世 |
| 第二世 | 晚年被太初古礦七大古皇圍攻，又遭不死天皇偷襲重傷。小松背葉凡到地球成仙地 |
| 第三世 | 在九十九龍山成仙池中涅槃升華。借麒麟古皇長生法及人形不死藥開啟第四世 |
| 第四世 | 繼續蓄力蛻變 |
| 第五世 | 萬道熔一，化生混沌體，開創《混沌經》 |
| 第六世 | 斬去混沌體，返本還源（荒古聖體 -> 混沌體 -> 凡體） |
| 第七世 | 大夢萬古（走古法修煉） |
| 第八世 | 化為凡體。八世末紅塵為仙，成就紅塵仙境界，長生不朽 |
| 第九世 | 率領天庭諸強征戰仙路 |

### 16. 最終大戰與結局

- 第九世，葉凡、狠人大帝、無始大帝、段德四尊紅塵仙聯手
- 征戰仙路，擊殺不死天皇
- 斬殺帝尊（最終BOSS，神話時代天庭之主）
- 成功打開仙域
- 天庭舉教飛仙，率眾進入仙域
- 黑皇淚流滿面，再次見到無始大帝

## 二、修煉體系

### 五大秘境（核心修煉體系）

#### 1. 輪海秘境
- **苦海**：修煉臍下生命之輪，開闢苦海
- **命泉**：涌出命泉
- **神橋**：架設神橋
- **彼岸**：到達彼岸
- 特徵：修煉根基，開闢生命本源

#### 2. 道宮秘境
- **心之神藏**
- **肝之神藏**
- **脾之神藏**
- **肺之神藏**
- **腎之神藏**
- 特徵：修煉五臟，開啟五大神藏

#### 3. 四極秘境
- 四重天，修煉四肢接連大道
- 涉及空間之力
- 特徵：肉身蛻變，掌握空間力量

#### 4. 化龍秘境
- 以人體脊柱為對象修行
- 九個小境界（九變）
- 特徵：化龍九變，壽命可活千年

#### 5. 仙台秘境
- 修煉腦內淨土
- 與前四大秘境存在本質區別
- 分層天進階

### 特殊說明
- **大帝**：人族成道者在荒古時代的稱呼
- **古皇**：妖族成道者在太古時代的稱呼
- **天尊**：神話時代成道者的稱呼
- 三者本質相同，只是時代不同的稱謂
- **天心印記**：成帝者掌控的印記，象徵人道巔峰，與天道法則融合（葉凡例外，他凌駕天道之上）
- **大帝果位**：同一時期宇宙只認可一位大帝，是人道巔峰的象徵
- **大成聖體**：準帝九層巔峰的聖體狀態，戰力堪比凡體另類成道
- **紅塵仙**：超越大帝的存在，通過多世蛻變在紅塵中成仙

## 三、重要角色

### A. 主角與核心同伴

#### 葉凡（葉天帝）
- **體質**：荒古聖體（後經歷混沌體、凡體的蛻變）
- **功法**：《天帝經》（包含《道經》《西皇經》《四極經/恒宇經》《太皇經》《太陽聖經》《太陰聖經》《聖體秘術》，每個秘境最強修煉之法的合集）、《混沌經》（第五世自創）
- **秘術**：九秘中的六秘（皆、兵、鬥、者、行、前）、虛空大手印、六道輪迴拳、八步滅道
- **帝兵**：天帝鼎（玄黃母氣煉製，後融合萬物母氣鼎化為無上仙器）
- **其他法寶**：誅仙四劍、荒塔、玉淨瓶
- **妻子**：姬紫月（正妻）、晨溪（生下女兒葉依水）
- **弟子**：葉瞳、小松、花花等
- **成就**：建立天庭、平定禁區、終結黑暗動亂、舉教成仙

#### 龐博
- 葉凡大學時最好的朋友，壯碩魁偉，直率義氣
- 到達北斗後被青帝十九代孫附體，後葉凡與黑皇鎮壓老妖神識
- 重新掌控身軀後取得妖帝古經和青蓮法寶
- 習得妖帝九斬和天妖八式

#### 段德（曹雨生 / 渡劫天尊 / 冥尊 / 冥皇 / 源帝）
- 前世為亂古紀元的曹雨生，是荒天帝石昊最好的朋友
- 在神話時代初期復甦，此後以多個身份登場：
  - 第一世：渡劫天尊（神話時代天尊輝煌傳說的開創者，留下《渡劫天功》）
  - 第二世：以屍道、源術雙重證道，建立地府
  - 第五世：化名段德，猥瑣盜墓人形象，與葉凡相識
- 精通源術，是源天師一脈的傳承者
- 九道輪迴印齊聚，熔煉為一體，紅塵成仙
- 與葉凡、狠人、無始聯手誅殺帝尊

#### 黑皇（大黑狗）
- 比猛虎更大，如公牛般強壯，尾巴半禿
- 原為一條重傷垂死的黑狗，被無始大帝救活收養
- 無始大帝為其煉出神源液封在紫山中，十數萬年後隨葉凡離開
- 性格憊懶無恥卻重情重義，一直叫囂「收人寵」
- 精通陣法，在聖崖之行中負責鋪設陣紋
- 對無始大帝忠心耿耿，大結局淚流滿面再見無始大帝

#### 姬紫月
- 姬家小姐，出場年齡十七歲
- 被葉凡劫持一同經歷青銅古殿歷險
- 在孔雀王之亂中被追殺，又與葉凡相遇
- 被葉凡護送回姬家，感情在生死磨難中逐漸升溫
- 最終成為葉凡正妻，與葉凡相伴成仙

## 六、世界觀核心設定

### A. 天道的運作規則
- 天道是宇宙的根本法則，掌控萬物運行
- 大帝成道需獲取**天心印記**，與天道融合
- 天道會壓制特定存在（如聖體），降下先天道圖阻止
- 葉凡是例外 -- 他不認可天道，將天心印記凌駕於天道之上

### B. 大帝不能共存的法則
- **核心規則**：同一時期宇宙只能有一位大帝存在
- 大帝的法則會壓制萬道，使後輩難以成帝
- 這就是「大帝不能共存」-- 一位大帝在世時，其他人無法成道
- 每數萬載出現一位大帝鎮壓當世（荒古時代以前的規律）
- 荒古後天地巨變，法則有缺，斬道之上的修行變得極其艱難

### C. 仙界的設定
- **仙域**：真正的仙人世界，存在於主宇宙之外
- **成仙路**：每百萬年開啟一次（大帝壽命僅一兩萬年，無法等到）
- 因此大帝晚年自斬修為，封印仙源中沉睡，等待成仙路開啟 -- 這就是禁區至尊的由來
- **仙王**、**仙帝**：仙域中更高的境界
- **紅塵仙**：不經仙路，在紅塵凡世中成仙，需多世蛻變，成就後長生不朽

### D. 荒古大帝、太古大帝（古皇）的區別
- **神話時代**：成道者稱「天尊」，代表：帝尊領銜的九大天尊
- **太古時代**：成道者稱「古皇」，代表：不死天皇領銜的眾多妖族古皇。人族弱小
- **荒古時代**：成道者稱「大帝」，代表：狠人、虛空、恒宇、無始等。人族極度璀璨
- **荒古前與荒古後**（以無始大帝為分界）：荒古前每數萬載出一位大帝；荒古後天地巨變，法則有缺，修行極難
- 本質相同，只是時代不同，三者戰力等級一致

### E. 禁區至尊的來歷
- 禁區至尊 = 自斬一刀的天尊 / 古皇 / 大帝
- 他們壽元將盡，不甘死去，又等不到成仙路開啟
- 選擇自斬修為、用仙源封印自身、在禁區沉睡
- 但維持生機需要吞噬生靈 -- 這就是**黑暗動亂**的根源
- 每隔一段時間，禁區至尊會蘇醒發動黑暗動亂，屠戮眾生續命
